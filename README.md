# docfly

> One command turns a Markdown folder into a beautiful doc site, with live preview.

```bash
pip install docfly
docfly ./docs
```

- **Live preview** — edit Markdown, browser refreshes instantly
- **Code highlighting** — Pygments, 50+ languages
- **Dark/light theme** — toggle in one click
- **Nested navigation** — mirrors your folder structure
- **Front-matter** — title, order, and more

## Quick start

```bash
cd myproject
docfly docs/ --open
```
